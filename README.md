## 开始使用

这是一个支持多种多模态大模型的 AI 对话的拓展，支持文件、图像、网表输入。

1. 下载本拓展，在 **嘉立创EDA** 或 **EasyEDA** 专业版中安装并启用**外部交互**权限

2. 在原理图页面中，点击顶部菜单的 `Copilot`，即可打开本拓展

3. 在正式使用前需要先配置模型，点击 `模型` 选择模型，点击 `设置` 进行 API 配置
